Load a skill to get detailed instructions for a specific task.

- Call this tool with no `name` to list available skills.
- Use `query` to filter skills by task keywords before loading one.
- Call this tool with a specific `name` to load that skill's `SKILL.md` content.
- Set `include_diagnostics` to show invalid or duplicate skill files discovered during scanning.
- Skills are Markdown guides discovered from `.openagent`, `.opencode`, and `.claude` skill directories.
